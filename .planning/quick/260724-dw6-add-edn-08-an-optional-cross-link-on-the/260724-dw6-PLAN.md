<objective>
Implement EDN-08: add an OPTIONAL, UNIDIRECTIONAL cross-link on the édition detail page pointing to a related Portfolio gallery, where one exists — via a new optional `relatedGallery` reference field on the `edition` Sanity schema. The concrete real-world case (confirmed with Romane in Phase 11, PROJECT.md Key Decisions): the "Rebut" édition (printed book/zine) and the "Rebut" Portfolio gallery are the same underlying subject, and a visitor should be able to navigate from the édition to the gallery.

The field ships EMPTY on every édition — actually populating the "Rebut" édition's reference is a separate content-authoring step for Florian/Romane in Sanity Studio, NOT part of this plan (no seed content, no fixture writes to the live dataset).

Scope guardrails (from the task constraints):
- UNIDIRECTIONAL only (édition -> gallery). Do NOT add a reverse reference or any UI on the gallery side, and do NOT modify getGalleries()/getGallery() or the Gallery interface.

- No new dependencies. Do NOT touch sanity/schemas/structure.ts (a reference field renders in the default form).
