## Accomplishments
- `LanguageSwitcher.astro`: replaced all three hardcoded `#1a1a1a` values — switcher links now render pink (`var(--color-accent)`), separator uses `var(--color-ink)`, focus ring is a single pink outline; `.is-current` weight-600 + underline emphasis preserved
- `GalleryCard.astro`: title converted from fluid `clamp(2.5rem, 12cqi, 6.5rem)` + `font-weight: 900` to the fixed Display role (`var(--text-display-size)` 32px / `var(--weight-semibold)` / `var(--font-display)` / line-height 1.2); dropped `container-type: inline-size`; focus ring simplified from the old double ink+accent ring to a single pink ring
- `Lightbox.astro`: replaced the hardcoded `rgba(20, 18, 19, 0.96)` scrim (the decimal-RGB equivalent of Phase 2's superseded Woodsmoke `#141213`) with `rgba(26, 26, 26, 0.96)` (the new ink token's equivalent); counter's `font-weight: 300` → `var(--weight-regular)`; native `<dialog>`/`showModal()` architecture, focus containment, keyboard/touch handlers untouched
- `GalleryGrid.astro`: read and confirmed it already only references `var(--color-dominant)` — no change needed
- Both `src/pages/galleries/[slug].astro` and `src/pages/en/galleries/[slug].astro`: replaced the old pink inline-block header panel with a `70vh`/`min-height: 420px` full-bleed hero (`fullSizeUrl(gallery.images[0])`) + the exact dark gradient scrim (`linear-gradient(to top, rgba(0,0,0,.65) 0%, rgba(0,0,0,.15) 45%, rgba(0,0,0,0) 70%)`) + a white, absolutely-positioned bottom-left Display-role `<h1>` + a single white top-left back-link; both pages now invoke `<BaseLayout ... headerVariant="transparent">`; thumbnail focus rings simplified to a single pink ring to match the site-wide convention

- **Fix:** Reworded the comment to reference `var(--color-ink)` instead of the literal hex.
- **Files modified:** `src/components/Lightbox.astro`
- **Verification:** `grep -riE "#1a1a1a|#141213|F0E7E4|E4D9D0|F92D97" src/components/{GalleryCard,GalleryGrid,Lightbox,LanguageSwitcher}.astro` returns zero matches
- **Committed in:** `fcec401`
