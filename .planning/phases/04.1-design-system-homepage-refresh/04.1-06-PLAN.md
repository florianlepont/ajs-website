<objective>
Close the D-01 "full site rebrand" gap on the eight already-shipped content pages that Plans 03/05 do not touch: the About page (FR/EN), both legal pages (mentions-légales + confidentialité, FR/EN), and the Contact page (FR/EN). Their colors already repaint correctly via CSS-variable inheritance from BaseLayout (Plan 04.1-01), but their typography still carries the Phase-2 system — fluid `clamp()` Display sizing with `container-type`/`cqi`, `font-weight: 900` headings, and `font-weight: 300` body copy. Convert every h1/h2/p (and the Contact page's page-level `.contact-page__lead`) in these files to the fixed Display/Heading/Body roles, using the exact conversion pattern already applied to `galleries/index.astro` (Plan 04.1-03 Task 3) and `GalleryCard.astro` (Plan 04.1-05 Task 1).

Note: the Contact page's *form island* (`src/components/ContactForm.astro`) is handled separately by Plan 04.1-03; this plan touches only the page-wrapper files `src/pages/contact.astro`/`src/pages/en/contact.astro`, which carry their own page-level `<h1>Contact</h1>` heading + lead paragraph with a `.contact-page` `<style>` block still on the superseded Phase-2 type system.

Additionally (checker-identified D-06 gap): `src/pages/404.astro` — also already shipped — hardcodes its link color to the old ink hex in its scoped `<style>` block (`.not-found a { color: #1a1a1a; /* Accent */ }`), so its "Retourner à l'accueil" / "Return home" body-copy links never pick up the D-06 pink-link rule despite inheriting everything else from BaseLayout's tokens. Task 4 repaints that one declaration to `var(--color-accent)`.

Purpose: Without this, About/mentions-légales/confidentialité/Contact keep the old bold-900/fluid-clamp type system while every other page moves to the new fixed-32px/400-600 system — producing exactly the "visually split site before launch" D-01 was written to prevent. And without the 404 fix, one shipped page's links stay ink instead of pink, violating the site-wide D-06 link rule.
Output: Eight repainted content pages on the new fixed type roles, plus the 404 page's links on the accent token.
</objective>

- h1/h2 and p are ALREADY on the correct fixed Heading/Body values (20px/600 and 16px/400) — do not touch them.
