Primary/secondary action button — filled ink block or outline, no brand color yet.

```jsx
<Button variant="primary">Envoyer</Button>
<Button variant="secondary">Annuler</Button>
```

Variants: `size="sm"` for compact/inline placements. `disabled` dims to 40% opacity. Reserve `variant="primary"` for the single most important action on a screen (e.g. contact-form submit).
